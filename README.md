## 插件简介

输出属性快捷设置（OutputQuickSettings）是一款超紧凑型的Blender渲染输出设置工具，专为MMD（MikuMikuDance）工作流设计。支持多版本Blender兼容，提供一键式渲染参数设置和时间线帧设置功能。

## 功能特性

### 🎬 渲染引擎设置

- **引擎快速切换**
  - Cycles 渲染引擎
  - EVEEL 渲染引擎（自动适配不同Blender版本）
  - 显示当前引擎状态

### 📊 采样设置

| 类型     | 预设值                | 说明               |
| -------- | --------------------- | ------------------ |
| 渲染采样 | 16 / 32 / 64 + 自定义 | 控制最终渲染质量   |
| 视口采样 | 16 / 32 / 64 + 自定义 | 控制3D视口预览质量 |

### 🖼️ 分辨率预设

一键设置常用分辨率：

- **720p**: 1280 × 720
- **1080p**: 1920 × 1080
- **2K**: 2048 × 1080
- **4K**: 3840 × 2160

支持横屏/竖屏切换

### ⏱️ 帧率设置

支持常用帧率快速切换：

- 30 FPS
- 60 FPS
- 120 FPS

### ⏩ 时间线帧设置

快速设置当前帧和起始帧：

- 预设按钮：0帧 / 5帧 / 30帧
- 自定义输入框（支持0-100000范围）
- 实时显示当前帧和起始帧状态

### 📁 输出设置

- **输出路径**：一键浏览和打开输出文件夹
- **文件格式**：PNG / JPEG / EXR / 视频（FFMPEG）
- **颜色模式**：BW / RGB / RGBA
- **色深**：8位 / 16位
- **压缩率**：0-100（PNG专用）

### 🎥 视频编码（FFMPEG格式）

- H264
- H265
- MPEG4

### 🚀 快捷渲染

- 一键渲染动画按钮

## 安装步骤

### 方法一：从ZIP文件安装

1. 下载插件的ZIP压缩包
2. 打开 Blender
3. 进入 `编辑 > 首选项 > 插件`
4. 点击右上角的 `安装` 按钮
5. 选择下载的ZIP文件
6. 勾选启用插件

### 方法二：手动安装

1. 解压插件文件夹
2. 将 `OutputQuickSettings` 文件夹复制到 Blender 的插件目录：
   - Windows: `C:\Users\你的用户名\AppData\Roaming\Blender Foundation\Blender\4.5\scripts\addons\`
   - macOS: `~/Library/Application Support/Blender/4.5/scripts/addons/`
   - Linux: `~/.config/blender/4.5/scripts/addons/`
3. 打开 Blender，进入 `编辑 > 首选项 > 插件`
4. 搜索 "输出属性快捷设置" 并启用

## 使用指南

### 面板位置

插件面板位于：**视图3D > 属性面板 > MMD > 输出快捷设置**

### 快速操作流程

1. **选择渲染引擎**：点击 Cycles 或 EVEEL
2. **设置采样**：选择预设值或输入自定义数值
3. **选择分辨率**：点击预设分辨率或切换横竖屏
4. **设置帧率**：选择目标帧率
5. **设置时间线**：点击预设帧数或输入自定义帧
6. **配置输出**：设置输出路径和文件格式
7. **开始渲染**：点击"渲染动画"按钮

### 状态栏

面板底部显示当前所有设置的汇总信息：

- 当前引擎和采样设置
- 分辨率和方向
- 帧率和帧范围
- 文件格式和输出路径
- 色彩配置

## 系统要求

| 项目        | 最低要求                |
| ----------- | ----------------------- |
| Blender版本 | 2.93.0 或更高           |
| 操作系统    | Windows / macOS / Linux |

**支持 Blender 版本**：

- Blender 2.93 LTS
- Blender 3.x
- Blender 4.x（含4.5）

## 已知问题与限制

1. **采样设置限制**：自定义采样值范围为1-10000
2. **帧值限制**：自定义帧值范围为0-100000
3. **视频格式**：FFMPEG格式时显示编码选项，图像格式时显示颜色设置
4. **路径验证**：打开输出文件夹时自动创建不存在的目录
