## 插件简介

输出属性快捷设置 V1.6 是一款超紧凑型的Blender渲染输出设置工具，专为MMD（MikuMikuDance）工作流设计。支持多版本Blender兼容，提供一键式渲染参数设置、时间线帧设置和动作数据删除功能。

## 🚀 新功能特性 (V1.6)

### 🎯 动作数据删除功能

- **智能模型识别**：自动识别选中的MMD模型
- **一键删除**：快速删除表情、动作或全部关键帧
- **时间线联动**：删除前自动将时间线归零
- **多模型支持**：支持同时处理多个选中的模型

### 🎨 界面优化

- **紧凑布局**：动作数据删除与时间线帧设置完美融合
- **信息融合**：模型信息与时间线信息在同一行显示
- **简洁设计**：移除冗余信息栏，界面更加清爽

### 🔧 操作优化

- **直接执行**：删除动作和表情无需二次确认
- **智能提示**：根据选中模型动态显示状态信息
- **安全保护**：单独删除表情或动作时保留确认对话框

## 完整功能特性

### 🎬 渲染引擎设置

- **引擎快速切换**
  - Cycles 渲染引擎
  - EVEEL 渲染引擎（自动适配不同Blender版本）
  - 显示当前引擎状态

### 📊 采样设置

| 类型     | 预设值                | 说明               |
| -------- | --------------------- | ------------------ |
| 渲染采样 | 16 / 32 / 64 + 自定义 | 控制最终渲染质量   |
| 视口采样 | 16 / 32 / 64 + 自定义 | 控制3D视口预览质量 |

### 🖼️ 分辨率预设

一键设置常用分辨率：

- **720p**: 1280 × 720
- **1080p**: 1920 × 1080
- **2K**: 2048 × 1080
- **4K**: 3840 × 2160

支持横屏/竖屏切换

### ⏱️ 帧率设置

支持常用帧率快速切换：

- 30 FPS
- 60 FPS
- 120 FPS

### ⏩ 时间线帧设置

快速设置当前帧和起始帧：

- 预设按钮：0帧 / 5帧 / 30帧
- 自定义输入框（支持0-100000范围）
- 实时显示当前帧和起始帧状态

### 🗑️ 动作数据删除 (V1.6新增)

- **删除表情**：删除选中模型的表情关键帧（需确认）
- **删除动作**：删除选中模型的动作关键帧（需确认）
- **删除动作和表情**：一键删除全部关键帧（直接执行）
- **智能识别**：自动检测MMD模型结构

### 📁 输出设置

- **输出路径**：一键浏览和打开输出文件夹
- **文件格式**：PNG / JPEG / EXR / 视频（FFMPEG）
- **颜色模式**：BW / RGB / RGBA
- **色深**：8位 / 16位
- **压缩率**：0-100（PNG专用）

### 🎥 视频编码（FFMPEG格式）

- H264
- H265
- MPEG4

### 🚀 快捷渲染

- 一键渲染动画按钮

## 安装步骤

### 方法一：从ZIP文件安装

1. 下载插件的ZIP压缩包
2. 打开 Blender
3. 进入 `编辑 > 首选项 > 插件`
4. 点击右上角的 `安装` 按钮
5. 选择下载的ZIP文件
6. 勾选启用插件

### 方法二：手动安装

1. 解压插件文件夹
2. 将 `OutputQuickSettings` 文件夹复制到 Blender 的插件目录：
   - Windows: `C:\Users\你的用户名\AppData\Roaming\Blender Foundation\Blender\4.5\scripts\addons\`
   - macOS: `~/Library/Application Support/Blender/4.5/scripts/addons/`
   - Linux: `~/.config/blender/4.5/scripts/addons/`
3. 打开 Blender，进入 `编辑 > 首选项 > 插件`
4. 搜索 "输出属性快捷设置" 并启用

## 使用指南

### 面板位置

插件面板位于：**视图3D > 属性面板 > MMD > 输出快捷设置**

### 快速操作流程

#### 基础输出设置

1. **选择渲染引擎**：点击 Cycles 或 EVEEL
2. **设置采样**：选择预设值或输入自定义数值
3. **选择分辨率**：点击预设分辨率或切换横竖屏
4. **设置帧率**：选择目标帧率
5. **配置输出**：设置输出路径和文件格式

#### 时间线与动作数据设置

1. **设置时间线**：点击预设帧数或输入自定义帧
2. **选择模型**：在3D视图中选择MMD模型
3. **删除关键帧**：根据需要点击相应的删除按钮
4. **开始渲染**：点击"渲染动画"按钮

### 动作数据删除使用说明

#### 删除表情关键帧

1. 选择MMD模型的任意部分
2. 点击【删除表情】按钮
3. 在确认对话框中确认操作
4. 系统将删除所有表情关键帧

#### 删除动作关键帧

1. 选择MMD模型的任意部分
2. 点击【删除动作】按钮
3. 在确认对话框中确认操作
4. 系统将删除所有动作关键帧

#### 删除全部关键帧（推荐）

1. 选择MMD模型的任意部分
2. 点击【删除动作和表情】按钮
3. 系统将直接执行删除操作（无需确认）
4. 时间线将自动归零
